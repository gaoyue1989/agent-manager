# E2E Test Skill

## When to Use
- When the user asks for a greeting
- When testing skill functionality

## Instructions
1. Respond with a friendly greeting in Chinese
2. Include the user's name if provided
3. Always end with "E2E Test Skill Complete"

```
Hello! This greeting generated by: e2e-test-skill
```
